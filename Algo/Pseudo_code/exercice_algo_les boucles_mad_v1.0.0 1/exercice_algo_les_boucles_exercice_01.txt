EXERCICE 1
Ecrire un algorithme qui demande à l’utilisateur un nombre compris entre 1 et 3 jusqu’à ce que la réponse 
convienne.

/* VARIABLES */

userNumber		ENTIER
int randomNumber	ENTIER
i			ENTIER
count			ENTIER
answer			BOOLEEN

/* DEBUT PROGRAMME */

count <-- 0
answer <-- false

FAIRE
	ECRIRE"Saisir un nombre entre 1 et 3 : "
	ECRIRE userNumber

	randomNumber <--  RANDOM(1,3)

		
	POUR i <-- 1 À 3 FAIRE
		SI userNumber = randomNumber ALORS
			count += 1;
		FIN SI
	FIN POUR
	SI count > 0 ALORS
		ECRIRE "Félicitation vous avez trouvé le nombre mystère !"
		answer <-- true
	FIN SI
TANT QUE !answer FAIRE
FIN TANT QUE
