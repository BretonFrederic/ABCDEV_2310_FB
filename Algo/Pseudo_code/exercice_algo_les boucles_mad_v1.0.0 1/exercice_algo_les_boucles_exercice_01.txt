EXERCICE 1
Ecrire un algorithme qui demande à l’utilisateur un nombre compris entre 1 et 3 jusqu’à ce que la réponse 
convienne.

/* VARIABLES */

userNumber		ENTIER
answer			BOOLEEN

/* DEBUT PROGRAMME */

answer <-- false

FAIRE
	ECRIRE"Saisir un nombre entre 1 et 3 : "
	ECRIRE userNumber

	SI userNumber < 1 ET userNumber > 3 ALORS
		ECRIRE "La réponse ne convient pas."
	SINON
		ECRIRE "La réponse convient."
		answer = true
	FIN SI
TANT QUE !answer FAIRE
FIN TANT QUE
