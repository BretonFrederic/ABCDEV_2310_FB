import java.util.Scanner;

public class MyClass {
    public static void main(String args[]) {
        /* VARIABLES */
        
        double[] scoreCandidat = {0,0,0,0};
        double[] numeroCandidat = {1,2,3,4};
        
        /* DEBUT PROGRAMME */
        Scanner score = new Scanner(System.in);
        
        System.out.println("Saisir le score du candidat 1 : ");
        scoreCandidat[0] = score.nextDouble();
        System.out.println("Saisir le score du candidat 2 : ");
        scoreCandidat[1] = score.nextDouble();
        System.out.println("Saisir le score du candidat 3 : ");
        scoreCandidat[2] = score.nextDouble();
        System.out.println("Saisir le score du candidat 4 : ");
        scoreCandidat[3] = score.nextDouble();
        
        System.out.println("Résultat du premier tour : ");
        
        for(int i = 0; i < (scoreCandidat.length-1); i++){
            if(scoreCandidat[i] > scoreCandidat[i+1]){
                double tempScore = scoreCandidat[i];
                double tempNumero = numeroCandidat[i];
                scoreCandidat[i]=scoreCandidat[i+1];
                numeroCandidat[i]=numeroCandidat[i+1];
                scoreCandidat[i+1]=tempScore;
                numeroCandidat[i+1]=tempNumero;
            }
        }
        
        for(int i = (scoreCandidat.length-1); i > -1; i--){
            System.out.println("Le candidat numéro " + numeroCandidat[i] + " a obtenu " + scoreCandidat[i]);
        }
        
        if(scoreCandidat[3] > 50){
            System.out.println("Le candidat " + numeroCandidat[3] + " est élu au premier tour");
        }
    }
}

Les élections législatives, en Guignolerie Septentrionale, obéissent à la règle suivante :
• lorsque l'un des candidats obtient plus de 50% des suffrages, il est élu dès le premier tour.
• en cas de deuxième tour, peuvent participer uniquement les candidats ayant obtenu au moins 
12,5% des voix au premier tour.
Vous devez écrire un algorithme qui permette la saisie des scores de quatre candidats au premier tour. Cet 
algorithme traitera ensuite le candidat numéro 1 (et uniquement lui) : il dira s'il est élu, battu, s'il se trouve 
en ballottage favorable (il participe au second tour en étant arrivé en tête à l'issue du premier tour) ou 
défavorable (il participe au second tour sans avoir été en tête au premier tour).
